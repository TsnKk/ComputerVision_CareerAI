อุปกรณ์เป็นแนว
1.openCV ใช้เปิดกล้อง-ปิดกล้อง
2.mediapipe สำหรับวิเคราะห์สีหน้า (optional ทำทีหลัง)
3.อะไรสักอย่างสำหรับทำ STT กับ TTS
4.streamlit สำหรับแสดงข้อมูลเป็น dashboard
5.pandas สำหรับเก็บและวิเคราะห์ข้อมูล สรุปข้อมูล คำนวณข้อมูลสำหรับใช้แสดงผลใน streamlit
6.tensorflow สำหรับทำ ai วิเคราะห์สีหน้า/ใบหน้าผู้สมัคร (optional ทำทีหลัง)
7.FastAPI สำหรับเชื่อมต่อ API google gemini

ตอนนี้การทำงานของแอปเป็นแบบนี้คล่าวๆ
1.ผู้ใช้ copy Job Description มาจากในเน็ตเอามาวาง
2.gemini api ทำการวิเคราะห์ Job Description และสร้างคำถามสัมภาษงานมา แสดงคำถาม 10 ข้อที่สร้างมาพร้อมปุ่ม start สำหรับซ้อมสัมภาษ
3.เมื่อกด start openCV เปิดกล้องจับใบหน้า mediapipe วิเคราะห์ใบหน้า
4.ใช้ TTS อ่านคำถามที่สร้างขึ้นมา และรอให้ผู้ใช้ตอบก่อนแล้วค่อยอ่านคำถามถัดไป และก็มี STT เพื่อจำคำตอบผู้ใช้ไว้แสดงตอนจบ
5.เมื่อครบสิบคำถามจะมีการปิดกล้อง
6.สรุปผลออกมาเป็น dashboard ผ่าน streamlit ประมาณแบบว่า คำถามนี้ผู้ใช้ตอบอะไรไป คำตอบมีความชัดเจนแค่ไหน มีคะแนนทางด่านสีหน้ายิ้มแย้มเป็น percent 1-100
ใช้ gemini api ในการเอาคำตอบของผู้ใช้งานไปสรุป + ให้ feedback

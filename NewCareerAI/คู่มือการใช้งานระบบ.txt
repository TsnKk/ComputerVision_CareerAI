** คู่มือการใช้งานระบบ Career AI **

ไลบรารีที่จำเป็น:
    1. pyttsx3 - สำหรับแปลงข้อความเป็นเสียง
    2. SpeechRecognition - สำหรับรับรู้เสียง
    3. pyaudio - สำหรับการบันทึกเสียง
    4. sounddevice - สำหรับอุปกรณ์เสียง
    5. numpy - สำหรับการคำนวณ

วิธีติดตั้ง: 
เข้าไปที่ไฟล์ requirements.txt แล้วใช้คำสั่ง pip install -r requirements.txt
ไฟล์ setup.sh เป็นการให้สิทธิ์เข้าถึง ใช้คำสั่ง chmod +x setup.sh

โครงสร้างไฟล์:
- stt_module.py = แปลงเสียงเป็นข้อความ (Speech to Text)
- ai_module.py = ประมวลข้อความและสร้างคำตอบ
- tts_module.py = แปลงข้อความเป็นเสียง (Text to Speech)
- memory.py = เก็บประวัติการสนทนาในไฟล์ JSON
- main.py = ไฟล์หลักสำหรับรันโปรแกรม (ใช้คำสั่ง python3 main.py)

หมายเหตุ: 
แนะนำให้ใช้ Python 3.13.5 หรือ 3.11 เนื่องจากโมเดลบางตัวอาจไม่รองรับเวอร์ชันใหม่กว่านี้
